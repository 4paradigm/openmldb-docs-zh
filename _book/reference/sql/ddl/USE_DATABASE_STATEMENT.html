USE数据库

USE DATABASE

Syntax

UseDatabaseStmt ::=
    'USE' DBName

DBName ::=
    Identifier

USE 语句可为用户会话选择当前数据库。

SQL语句模版

USE database_name;

Example:

创建一个数据库db1:

CREATE DATABASE db1;
-- SUCCEED: Create database successfully

CREATE DATABASE db2;
-- SUCCEED: Create database successfully

然后选择db1作为当前数据库：

USE db1;
-- SUCCEED: Database changed

创建两张表:

CREATE TABLE t1(col0 string);
-- SUCCEED: Create successfully

CREATE TABLE t1(col0 string);
-- SUCCEED: Create successfully

SHOW TABLES;
 -------- 
  Tables  
 -------- 
  t1      
  t2      
 --------

然后选择db2作为当前数据库，并查看当前库下的表：

USE db2;
-- SUCCEED: Database changed

SHOW TABLES;
 -------- 
  Tables  
 -------- 
0 row in set

相关语句

DROP DATABASE

SHOW DATABASES

SHOW TABLES
